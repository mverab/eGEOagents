# E-GEO

### Zero-Effort Generative Engine Optimization

> One command. Premium results. Rank higher in ChatGPT, Perplexity, and AI search engines.

---

## ✨ What is E-GEO?

E-GEO transforms your website content to rank higher in AI-powered search engines. No learning curve. No complex setup. Just results.

```
/geo https://yoursite.com/pricing
```

That's it. E-GEO analyzes, optimizes, and delivers production-ready content.

---

## 🚀 Quick Start

### 1. Install

Copy the `.claude/` folder to your project:

```bash
cp -r /path/to/egeo-claude-agents/.claude .
```

### 2. Activate

In Claude Code, switch to GEO mode:

```
/output-style geo-optimizer
```

### 3. Optimize

Run your first optimization:

```
/geo https://yoursite.com
```

---

## 📋 Commands

| Command | What it does |
|---------|--------------|
| `/geo <url>` | **Full pipeline** - Analyze, rank, rewrite, schema |
| `/geo:audit <url>` | Analysis only, no changes |
| `/geo:optimize <file>` | Optimize local content file |
| `/geo:batch <folder>` | Process entire folder |
| `/geo:report` | Generate executive report |
| `/geo:compete <query>` | Competitive analysis |

## 🤖 How It Works

E-GEO uses 4 specialized AI agents:

| Agent | Role |
|-------|------|
| **Analyzer** | Extracts content, scores GEO signals, identifies gaps |
| **Ranker** | Simulates AI-engine ranking, predicts positions |
| **Rewriter** | Optimizes content while preserving brand voice |
| **Indexer** | Generates schema markup and technical assets |

Claude Code orchestrates these automatically. You just run one command.
